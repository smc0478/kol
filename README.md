# 콜 (Kol) 프로그래밍 언어

[![Build Status](https://img.shields.io/badge/build-passing-brightgreen)](https://github.com/smc0478/kol)
[![License](https://img.shields.io/badge/license-MIT-blue)](https://github.com/smc0478/kol/blob/main/LICENSE)
[![Version](https://img.shields.io/badge/version-1.0.0-informational)](https://github.com/smc0478/kol)

'콜(Kol)'은 한국어의 자연스러운 문법을 그대로 코드로 옮겨, 누구나 쉽게 프로그래밍의 세계에 입문할 수 있도록 설계된 스크립트 언어입니다.

---

### "안녕하세요, 콜!"

```kol
"안녕하세요, 콜!"을 출력한다.
```

**실행:**
```bash
python3 콜.py hello.kol
```

**결과:**
```
안녕하세요, 콜!
```

---

## ✨ 주요 특징

- **💬 자연어 문법**: `"만약 ... 이면 다음"`과 같이 일상적인 한국어처럼 코드를 작성하세요.
- **🚀 직관적인 학습 곡선**: 복잡한 기호 대신 익숙한 단어를 사용하여 프로그래밍을 처음 접하는 사람도 빠르게 배울 수 있습니다.
- **⚙️ VM 기반 실행**: 바이트코드로 컴파일되어 가상 머신 위에서 동작하므로, 플랫폼에 구애받지 않습니다.
- **📚 풍부한 내장 기능**: 기본적인 산술 연산부터 자료구조 처리까지, 다양한 내장 함수를 바로 사용할 수 있습니다.

## 🚀 빠르게 시작하기

1.  **저장소 복제**
    ```bash
    git clone https://github.com/your-repo/kol.git
    cd kol
    ```

2.  **예제 파일 실행**
    ```bash
    python3 콜.py example/hello.kol
    ```

3.  **디버그 모드 실행**
    디버그 모드로 실행하려면 `--디버그` 또는 `-디` 옵션을 추가합니다.
    ```bash
    python3 콜.py example/hello.kol --디버그
    ```
