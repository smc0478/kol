# 연산자 (Operators)

'콜' 언어는 다양한 이항 연산자를 지원하여 값들을 비교하거나 논리 연산을 수행할 수 있습니다.

## 1. 비교 연산자

| '콜' 연산자 | 설명 | Python 등가 | 예시 |
| :---------- | :--- | :---------- | :--- |
| `... 이/가 ... 랑 같다` | 두 값이 같은지 비교합니다. | `==` | `1이 1이랑 같다` (참) |
| `... 이/가 ... 랑 다르다` | 두 값이 다른지 비교합니다. | `!=` | `1이 2랑 다르다` (참) |
| `... 이/가 ... 보다 크다` | 왼쪽 값이 오른쪽 값보다 큰지 비교합니다. | `>` | `5가 3보다 크다` (참) |
| `... 이/가 ... 보다 작다` | 왼쪽 값이 오른쪽 값보다 작은지 비교합니다. | `<` | `3이 5보다 작다` (참) |
| `... 이/가 ... 보다 크거나 같다` | 왼쪽 값이 오른쪽 값보다 크거나 같은지 비교합니다. | `>=` | `5가 5보다 크거나 같다` (참) |
| `... 이/가 ... 보다 작거나 같다` | 왼쪽 값이 오른쪽 값보다 작거나 같은지 비교합니다. | `<=` | `5가 5보다 작거나 같다` (참) |
| `... 이/가 ... 비슷하다` | 두 값이 유사한지 비교합니다. (타입 변환을 시도하여 비교) | 없음 | `"10"이 10이랑 비슷하다` (참) |
| `... 이/가 ... 비슷하지 않다` | 두 값이 유사하지 않은지 비교합니다. | 없음 | `"10"이 20이랑 비슷하지 않다` (참) |

## 2. 논리 연산자

| '콜' 연산자 | 설명 | Python 등가 | 예시 |
| :---------- | :--- | :---------- | :--- |
| `그리고` | 두 피연산자 모두 `참`일 때 `참`을 반환합니다. | `and` | `참 그리고 거짓` (거짓) |
| `또는` | 두 피연산자 중 하나라도 `참`일 때 `참`을 반환합니다. | `or` | `참 또는 거짓` (참) |

## 3. 비트 쉬프트 연산자

| '콜' 연산자 | 설명 | Python 등가 | 예시 |
| :---------- | :--- | :---------- | :--- |
| `... 이/가 ... 만큼 왼쪽으로 간 것` | 왼쪽 피연산자를 오른쪽 피연산자만큼 왼쪽으로 비트 쉬프트합니다. | `<<` | `1이 2만큼 왼쪽으로 간 것` (4) |
| `... 이/가 ... 만큼 오른쪽으로 간 것` | 왼쪽 피연산자를 오른쪽 피연산자만큼 오른쪽으로 비트 쉬프트합니다. | `>>` | `4이 2만큼 오른쪽으로 간 것` (1) |
