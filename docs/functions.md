# 함수 (Functions)

'콜' 언어에서 함수를 정의하고 호출하는 방법을 설명합니다.

## 1. 함수 정의

함수는 `함수 함수이름은 매개변수1과 매개변수2로 다음 ... 문단을 실행한다.` 형태로 정의합니다.
함수는 `결과 값은 ...이 된다. 그리고 끝난다.` 구문을 사용하여 값을 반환할 수 있습니다. 반환 구문이 없으면 `없음` (None)이 반환됩니다.

```kol
함수 팩토리얼을_구한다는 가로 다음
    만약 가가 0이랑 같다면 다음
        결과 값은 1이 된다. 그리고 끝난다.
    문단을 실행한다.
    결과 값은 가와 ((가를 1로 뺀 것)으로 팩토리얼을_구한 것)을 곱한 것이 된다. 
    그리고 끝난다.
문단을 실행한다.

함수 인사는 이름으로 다음
    이름과 "님 안녕하세요!"를 더한 것을 출력한다.
문단을 실행한다.

함수 바꾼다는 가와 나로 다음
    결과 값은 나가 된다. 그리고 끝난다.
문단을 실행한다.
```

## 2. 함수 호출

'콜' 언어에서는 함수 호출을 문장(Statement) 또는 표현식(Expression)으로 사용할 수 있습니다.

### 함수 호출 문장 (Call Statement)

함수의 반환 값을 사용하지 않고 단순히 함수를 실행할 때 사용합니다. `... 와/과 ... 을/를 .. 와/과 ... 으로/로 ... 한다/다.` 형태로 작성됩니다.

```kol
4로 팩토리얼을_구한 것을 출력한다.
"홍길동"으로 인사한다.
1을 2로 바꾼 것을 출력한다.
```

### 함수 호출 표현식 (Call Expression)

함수의 반환 값을 다른 표현식의 일부로 사용하거나 변수에 할당할 때 사용합니다. `... 한 것` 또는 `... 것` 형태로 작성됩니다.

```kol
합계는 10과 20을 더한 것이 된다.
합계를 출력한다.
```
