# 내장 함수 (Built-in Functions)

'콜' 언어는 기본적인 작업을 수행하기 위한 다양한 내장 함수를 제공합니다. 이 함수들은 별도의 선언 없이 바로 사용할 수 있습니다.

## 1. 산술 연산

| '콜' 함수 이름 | 설명 | Python 등가 |
| :------------ | :--- | :---------- |
| `더(a, b)` | `a`와 `b`를 더합니다. | `a + b` |
| `뺀(a, b)` | `a`에서 `b`를 뺍니다. | `a - b` |
| `곱(a, b)` | `a`와 `b`를 곱합니다. | `a * b` |
| `나눈(a, b)` | `a`를 `b`로 나눕니다. 정수 나눗셈의 경우 `//`와 동일하게 동작합니다. | `a / b` 또는 `a // b` |
| `나머지를_구(a, b)` | `a`를 `b`로 나눈 나머지를 반환합니다. | `a % b` |

## 2. 입출력

| '콜' 함수 이름 | 설명 | Python 등가 |
| :------------ | :--- | :---------- |
| `읽은()` | 사용자로부터 한 줄을 입력받아 문자열로 반환합니다. | `input()` |
| `출력(obj)` | 객체 `obj`를 표준 출력에 출력합니다. | `print(obj)` |

## 3. 타입 변환

| '콜' 함수 이름 | 설명 | Python 등가 |
| :------------ | :--- | :---------- |
| `문자열으로_변환(obj)` | 객체 `obj`를 문자열로 변환합니다. | `str(obj)` |
| `정수로_변환(obj)` | 객체 `obj`를 정수로 변환합니다. | `int(obj)` |
| `소수로_변환(obj)` | 객체 `obj`를 부동 소수점 숫자로 변환합니다. | `float(obj)` |
| `진리값으로_변환(obj)` | 객체 `obj`를 불리언 값으로 변환합니다. | `bool(obj)` |

## 4. 유틸리티

| '콜' 함수 이름 | 설명 | Python 등가 |
| :------------ | :--- | :---------- |
| `길이를_구(obj)` | 객체 `obj`의 길이를 반환합니다 (문자열, 배열, 딕셔너리 등). | `len(obj)` |
| `범위로_만든(start, end)` | `start`부터 `end-1`까지의 정수 리스트를 반환합니다. | `list(range(start, end))` |
| `문자로_만든(num)` | 정수 `num`에 해당하는 유니코드 문자를 반환합니다. | `chr(num)` |
| `정수로_만든(s)` | 문자 `s`의 유니코드 코드 포인트를 반환합니다. | `ord(s)` |
| `자른(obj, start, end)` | `obj`의 `start` 인덱스부터 `end` 인덱스 전까지의 슬라이스를 반환합니다. | `obj[start:end]` |
| `에서_찾는(obj, obj2)` | `obj2`가 `obj` 안에 있는지 확인합니다. | `obj2 in obj` |
| `종료(code)` | 프로그램 `code`와 함께 종료합니다. | `exit(code)` |
| `문제가_발생(str)` | `str` 메시지와 함께 예외를 발생시킵니다. | `raise Exception(str)` |

## 5. 논리 및 비트 연산

| '콜' 함수 이름 | 설명 | Python 등가 |
| :------------ | :--- | :---------- |
| `부정(a)` | `a`의 논리적 부정을 반환합니다. | `not a` |
| `반전(a)` | `a`의 비트 반전을 반환합니다. | `~a` |
| `논리곱(a, b)` | `a`와 `b`의 비트 논리곱을 반환합니다. | `a & b` |
| `논리합(a, b)` | `a`와 `b`의 비트 논리합을 반환합니다. | `a | b` |
| `베타적_논리합(a, b)` | `a`와 `b`의 비트 베타적 논리합을 반환합니다. | `a ^ b` |

## 6. 배열 및 딕셔너리 조작

| '콜' 함수 이름 | 설명 | Python 등가 |
| :------------ | :--- | :---------- |
| `추가(arr, obj)` | 배열 `arr`의 끝에 `obj`를 추가합니다. | `arr.append(obj)` |
| `꺼낸(arr)` | 배열 `arr`의 마지막 요소를 제거하고 반환합니다. | `arr.pop()` |
| `키들을_가져온(d)` | 딕셔너리 `d`의 모든 키를 반환합니다. | `d.keys()` |
| `값들을_가져온(d)` | 딕셔너리 `d`의 모든 값을 반환합니다. | `d.values()` |
| `쪼갠(s, sep)` | 문자열 `s`를 `sep`을 기준으로 쪼개어 리스트로 반환합니다. | `s.split(sep)` |
| `합친(sep, arr)` | 배열 `arr`의 요소들을 `sep`으로 연결하여 문자열로 반환합니다. | `sep.join(arr)` |

```kol
출력("안녕하세요!")을 호출한다.
숫자는 읽은 것을 호출한 것이 된다.
길이는 "문자열"의 길이를_구한 것이 된다.
```
