# 데이터 타입 (Data Types)

'콜' 언어에서 지원하는 기본적인 데이터 타입들을 설명합니다.

## 1. 숫자 (Numbers)

- **정수 (Integer)**: `10`, `-5`, `0`
- **소수 (Float)**: `3.14`, `-0.1`, `2.0`

```kol
정수는 100이 된다.
소수는 3.141592가 된다.
```

## 2. 문자열 (Strings)

큰따옴표(`"`) 또는 작은따옴표(`'`)로 묶인 텍스트 데이터입니다.

```kol
이름은 "홍길동"이 된다.
메시지는 '안녕하세요, 콜 언어!'가 된다.
```

## 3. 불리언 (Booleans)

논리적인 `참` 또는 `거짓` 값을 나타냅니다.

```kol
참값은 참이 된다.
거짓값은 거짓이 된다.
```

## 4. 없음 (None)

값이 없음을 나타내는 특별한 값입니다. 다른 언어의 `null` 또는 `none`과 유사합니다.

```kol
빈값은 없음이 된다.
```

## 5. 배열 (Arrays)

대괄호(`[]`) 안에 `다음` 키워드로 구분된 요소들을 나열하여 생성합니다.

```kol
숫자_배열은 [1 다음 2 다음 3 다음 4]이 된다.
혼합_배열은 ["안녕" 다음 10 다음 참]이 된다.
빈_배열은 []이 된다.
```

## 6. 딕셔너리 (Dictionaries)

중괄호(`{}`) 안에 `키"는 "값` 형태로 키-값 쌍을 나열하고, `또` 키워드로 구분하여 생성합니다.

```kol
사람_정보는 {"이름"은 "김철수" 또 "나이"는 25}이 된다.
빈_딕셔너리는 {}이 된다.
```
