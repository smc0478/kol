# 구조체 및 클래스 (Structs and Classes)

'콜' 언어는 데이터와 기능을 캡슐화하기 위한 `구조체`와 `유형` (클래스)을 제공합니다.

## 1. 구조체 (Struct)

구조체는 데이터 필드만을 가지는 사용자 정의 데이터 타입입니다.

```kol
구조 사람은 다음
    변수 이름이 있다.
    변수 나이가 있다.
값을 가진다.

철수는 사람이 된다.
철수의 이름은 "철수다"가 된다.
철수의 이름을 출력한다. 
```

## 2. 유형 (Class)

유형(클래스)은 데이터 필드와 함께 메서드(함수)를 가질 수 있는 사용자 정의 데이터 타입입니다. 다른 언어의 클래스와 유사합니다.

```kol
유형 동물은 다음
    변수 이름이 있다.
    
    함수 소리를_낸다는 다음
        동물의 이름과 "이(가) 소리를 냅니다."를 더한 것을 출력한다.
    문단을 실행한다.
값을 가진다.

강아지는 동물이 된다.
강아지의 이름은 "바둑이"가 된다.
강아지 안에서 소리를_낸다. 
```

### 상속 (Inheritance)

클래스는 다른 클래스를 상속받을 수 있습니다. `유형 자식클래스는 부모클래스로 다음 ...` 형태로 작성합니다.

```kol
유형 강아지는 동물의 자식이고 다음
    변수 품종이 있다.
    
    함수 소리를_낸다는 자신으로 다음
        자신의 이름과 "의 품종은: "를 더한 것을 줄바꿈없이_출력한다.
        자신의 품종을 출력한다.
        자신의 이름과 "이(가) 소리를 냅니다."를 더한 것을 출력한다.
    문단을 실행한다.
값을 가진다.

바둑는 강아지가 된다.
바둑의 이름은 "바둑이"가 된다.
바둑의 품종은 "시고르자브종"이 된다.
바둑 안에서 소리를_낸다. 
```
